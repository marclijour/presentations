Fix a few details
Install Samba, the SMB le server, & create the expected folder with the right permissions
$ sudo su - # to become root
# apt-get install samba
# /etc/init.d/samba restart
# mkdir /var/lib/samba/usershares/
# chmod 1775 /var/lib/samba/usershares/
# chown root:sambashare /var/lib/samba/usershares/
# service samba restart # or try /etc/init.d/samba restart
# exit

Another potential x
In case your computer has a long name ( 15 characters)
Samba prefers short names (less than 15 characters). Add the following
line in /etc/samba/smb.conf, just after the line with server string. . . :
netbios name = Whatever
(Change Whatever by whatever you want your computer name to be.)
$ sudo su - # to become root
# gedit /etc/samba/smb.conf
# exit

Set a Work Group
Lets create our own SBDI group
You need to edit the cong le /etc/samba/smb.conf
$ sudo su - # to become root
# gedit /etc/samba/smb.conf
# exit
After workgroup = , change WORKGROUP for SBDI. Save the le and
restart Samba.
$ sudo /etc/init.d/samba restart
